
EmporiUm Bookstore Sales Analysis - Capstone 3

Dashboard Features

Page 1: Sales Overview
- KPI Cards: Total Sales, Total Quantity, Order Count, Average Order Value
- Line Chart: Sales trends over time
- Column Chart: Sales by state
- Stacked Column: Textbook vs General book sales by state
- Top 10 Table: Highest-performing states
- Interactive Slicers: Date, Category, Store ID

 Page 2: Detailed Analysis
- Donut Chart: Sales breakdown by product category
- Top 10 Books Table: Best-selling non-textbook books with authors
- Matrix: Regional and manager organizational hierarchy
- Bar Chart: Sales by region
- Scatter Plot: Quantity vs Sales relationship
- Interactive Slicers: Date, Region, Book Type
